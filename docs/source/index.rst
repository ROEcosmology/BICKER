Welcome to ``BICKER``'s documentation!
======================================

``BICKER`` is a package for predicicting the galaxy power spectrum and bispectrum with emulated perturbation theory kernels.

Installation
============

The package can be installed by cloning the repository and using ``pip``::


   https://github.com/ROEcosmology/BICKER.git
   cd BICKER
   pip install -e ./
